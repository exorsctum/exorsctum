Notes:
- Images update daily via GitHub Actions. First run appears after you add required secrets.
- Configure plugins and layout in `.github/workflows/metrics.yml`.

Setup:
- Edit `.github/workflows/metrics.yml` and fill the `env:` placeholders (user, AniList, Music, 16personalities, etc.).
- Add a `METRICS_TOKEN` repository secret (PAT) for the action to read data.
